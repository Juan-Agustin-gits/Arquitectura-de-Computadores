# Instrucciones de compilación 

El código fue testeado desde el SO Ubuntu 22.04.2 LTS. Linux.

1. Una vez descargado y descomprimido el archivo, abrir la terminal en la carpeta donde se encuentra tarea1.c

2. Escribir en terminal el comando: "gcc tarea1.c". Esto creará el ejecutable a.out.

3. Abrir el ejecutable con el comando "./a.out" para ver lo que compila el código C.
